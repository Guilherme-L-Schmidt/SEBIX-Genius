# SEBIX - Genius
Project for the development of a Genius game with Arduino.

## Code
The included `Genius_code.ino` is responsible for the game functions, and must be uploaded to the Arduino.

## Materials and prototype
- Arduino UNO
- PCB (or protoboard)
- 4 LEDs
- 2 10kΩ resistors
- 4 1kΩ resistors
- 4 buttons
